# predict-energy-consumption-datacamp
A Machine Learning exercise that predicts daily power consumption using supervised regression models.
--
This repository contains a short Machine Learning exercise from DataCamp, focused on predicting **daily power consumption** using supervised regression models.
The goal is to experiment with different models, evaluate their performance using **RMSE**, and compare predicted vs. actual consumption trends on the test dataset.

Project page: https://projects.datacamp.com/projects/2387

Two variables are produced as part of the assignment:

1. **selected_rmse** — the lowest RMSE achieved on the test set,

2. **trend_similarity** — whether the predicted values follow the same trend as the real data.
